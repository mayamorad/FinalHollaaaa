About This Project

DataVine 12 is a project driven by a passion for both data science and wine! We focused on predicting wine quality using machine learning, but made sure to keep the process enjoyable and creative. After all, who doesn’t enjoy a glass of wine while learning new skills? Our goal is to help both wine enthusiasts and professionals understand what makes a high-quality wine by exploring the relationship between characteristics like acidity, alcohol content, and pH. DataVine 12 uses machine learning to predict wine quality based on various physiochemical features. This project integrates data analysis, visualization, and predictive modeling to provide insights into what makes a high-quality wine, leveraging the power of machine learning algorithms.

  • Use machine learning with libraries like Scikit-learn.
  • Implement at least two of the following:
    • Python Pandas for data manipulation
    • Matplotlib for data visualization
  • Machine Learning: Scikit-learn
  • Data Manipulation: Pandas
