The Process

Here, we outline how our machine learning model was trained and tested to predict wine quality, including the dataset and methodologies used.

Data Preparation

We merged red and white wine datasets, added a wine_type column, standardized the numeric features, and exported a clean version for modeling and analysis.

Exploratory Analysis in Spark

Using Spark SQL, we analyzed average wine quality by type, alcohol distribution, and statistical summaries — confirming alcohol’s strong correlation with higher quality.

Machine Learning Modeling

We applied Random Forest, Logistic Regression, and SVM to classify wines by quality, achieving up to 89% accuracy without PCA, and slightly lower with PCA applied.

Model Insights

Feature importance highlighted alcohol, sulphates, and volatile acidity as key predictors of wine quality, confirming our hypothesis about the predictive value of these traits.
